# SQLServerAnalyzerFor1C
Анализ настроек, производительности и текущих запросов сервера для 1Сника 

Обработка и расширения для простого и удобного анализа настроек, нагрузки и проблем с SQL сервером с упором на использование оного для 1С. 

Анализ настроек сервера, текущих запросов на sql, ожиданий, конвертация запроса в 1С и рекомендации, где может тормозить.
Поддерживает MS SQL & PostgreSQL

Подробное описание на https://infostart.ru/1c/tools/2039548/
